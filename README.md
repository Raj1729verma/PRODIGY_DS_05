## 📌 Project Overview
This project analyzes a massive dataset of US traffic accidents (millions of records) to uncover spatial and temporal patterns. By examining the intersection of road conditions, weather, and time of day, this analysis identifies critical accident hotspots and contributing risk factors.

The insights generated from this project demonstrate real-world business value for **logistics planning, insurance risk modeling, and urban infrastructure development.**

## 🚀 Key Features

* **Big Data Processing:** Implemented memory-optimization techniques to efficiently process and clean a massive multi-year dataset without crashing system resources.
* **Geospatial Analysis:** Mapped complex geographic coordinates to visualize high-risk "accident hotspots" across different states and counties using interactive maps.
* **Multivariate & Time-Series Analysis:** Extracted trends based on time of day, seasonality, and weather conditions (e.g., how rain vs. snow impacts accident severity in different regions).
* **Interactive Dashboards:** Created visual representations that allow non-technical stakeholders to easily understand complex road safety metrics.

## 🛠️ Tech Stack
* **Language:** Python
* **Data Processing:** Pandas, NumPy (or PySpark for distributed processing)
* **Data Visualization:** Matplotlib, Seaborn, Plotly
* **Geospatial Mapping:** Folium, GeoPandas
* **Environment:** Jupyter Notebook

## 📊 Dataset
The data used in this project is the **US Accidents Dataset**, which covers 49 states of the US. 
* **Source:** [Kaggle - US Accidents](https://www.kaggle.com/datasets/sobhanmoosavi/us-accidents)
* *(Note: Due to the large file size, the dataset is not uploaded directly to this repository. Please download it from the link above and place it in the `data/` folder).*

## 💡 Business Use Cases Demonstrated
1. **Logistics & Supply Chain:** Optimizing routing software to avoid statistically dangerous intersections during specific weather conditions.
2. **Insurance Agencies:** Enhancing risk assessment models by factoring in highly granular spatial and temporal accident data.
3. **City Planning:** Providing data-driven recommendations for traffic law enforcement and road safety improvements.
